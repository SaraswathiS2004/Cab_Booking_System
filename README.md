Project : RideX
Tech Stack : Java

Features :
     
1. Admin Account Creation.
2. Role Based Access.
3. User Login Feature.
4. Nearest Cab ArrangeMent.
5. View Total Earnings.
     
Admin :
 
   1. Registration / login.
   2. View Accounts Details.
   3. View Drivers Details.
   4. View Cabs Current Position.
   5. View Cab Details.
   6. View UserTrip Details.
   7. View Feed Back Details
   8. View Cabs Total Earnings.
   9. View Individual Cab Earnings
   10. Log Out.
   11. Exit.

Users :

   1. Register / Login.
   2. Enter travel details.
   3. Check Available Cabs and Drivers.
   4. Feed Back Upload.
   5. Log Out.
   6. Exit.

Drivers :

      1. Registration / Login.
      2. Enter Cab Details.
      3. Enter Driver Details.
      4. Exit.
